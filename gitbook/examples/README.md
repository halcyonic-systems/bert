## Complexity Progression

The examples span a range of complexity scores using Simonian complexity calculations:

1. **Solar Panel** (15.6) - Engineered energy conversion
2. **Cell** (16.2) - Biological foundation  
3. **System Template** (18.4) - Universal principles
4. **BERT Platform** (19.7) - Self-referential software system
5. **Organization** (21.9) - Social coordination structure
6. **Ecosystem** (24.8) - Complex adaptive biological system
7. **LLM** (28.3) - Advanced artificial intelligence system

Each example demonstrates different aspects of systems science theory applied through BERT's visual modeling approach.
